__Учебный проект по автоматизации тестирования Petstore API__

Для установки всех зависимостей можно воспользоваться командой:

      pip install -r requirements.txt

Для корректной работы тестов может потребоваться VPN. 


<br>__Задание__

Автоматизировать проверку трёх сценариев.


<br>__Сценарий 1: Полный цикл работы с питомцем__

1. Добавить нового питомца с корректными данными
2. Загрузить фото питомца, проверить успешный статус код (200)
3. Получить данные питомца по ID, проверить, что они совпадают с ранее отправленными
4. Обновить статус питомца
5. Обновить имя питомца через форму
6. Получить имя и статус питомца по ID, проверить, что они обновились
7. Удалить питомца
8. Получить данные питомца по ID, питомец не должен быть найден (статус код 404)
9. Найти всех питомцев по статусу, проверить соответствие ответа модели


<br>__Сценарий 2: Управление заказами и инвентарём__

1. Получить инвентарь, проверить, что он содержит статусы 'available', 'pending' и 'sold'
2. Создать заказ с корректными данными
3. Получить данные заказа по ID, проверить, что они совпадают с ранее отправленными
4. Удалить заказ
5. Получить данные заказа по ID, заказ не должен быть найден (статус код 404)


<br>__Сценарий 3: Полный цикл управления пользователями__

1. Создать список из 5 пользователей, зафиксировать данные второго пользователя
2. Получить второго пользователя по username, проверить, что данные соответствуют отправленным
3. Создать одного нового пользователя с корректными данными
4. Получить пользователя по username, проверить, что данные соответствуют отправленным
5. Обновить телефон и email пользователя
6. Получить телефон и email пользователя по username, проверить, что они обновлены
7. Залогиниться под пользователем, проверить успешный статус код (200)
8. Разлогиниться, проверить успешный статус код (200)
9. Удалить пользователя
10. Получить пользователя по username, пользователь не должен быть найден (статус код 404)
